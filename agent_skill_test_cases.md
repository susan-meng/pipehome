## 一、测试概览

### 1.1 测试目标

验证执行工厂对外部 Skill 的注册、管理、读取、删除等功能符合 PRD 要求，确保渐进式加载、权限控制、删除补偿等核心机制正确实现。

### 1.2 测试范围

| 模块 | 覆盖功能 |
|------|----------|
| Skill 注册 | Zip 包注册、SKILL.md 内容注册 |
| Skill 管理 | 列表查询、详情查询、Zip 下载 |
| Skill 市场 | 市场列表、市场检索、市场详情 |
| Skill 使用 | Guide 读取、内部文件读取 |
| 删除补偿 | 正常删除、中间态管理、残留检测 |
| 安全权限 | 路径校验、权限控制、业务域隔离 |

### 1.3 优先级定义

- **P0**: 核心功能，阻塞发布
- **P1**: 重要功能，影响用户体验
- **P2**: 次要功能，边界场景
