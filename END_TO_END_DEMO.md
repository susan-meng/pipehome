```yaml
  path_patterns:
    - "vega/data-connection/**"
```

### apis.yaml

### GitHub Actions 触发条件

**Agent A 运行条件**:
```yaml
if: needs.commit-listener.outputs.has_changes == 'true' && 
    needs.commit-listener.outputs.need_cases == 'true'
```

只有当代码变更匹配到 Vega 路径且需要新增用例时才运行。

## 自动化脚本

脚本位置: `end_to_end_demo.sh`

使用方法:
```bash

```

脚本功能:
1. 自动创建示例代码

5. 自动合并到本地套件
6. 自动提交合并结果
